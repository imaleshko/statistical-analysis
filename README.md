# Статистичний аналіз даних

Вебзастосунок для побудови варіаційного ряду та обчислення основних статистичних характеристик вибірки

## Обчислення та вивід:

- варіаційний ряд
- частоти `nᵢ`, відносні частоти `wᵢ`
- накопичувальні частоти `nᵢ нак`, `wᵢ нак`
- полігон частот
- емпірична функція розподілу `F*(x)` — формула та графік
- мода, медіана, середнє значення
- дисперсія, середнє квадратичне відхилення
- початковий момент 2-го порядку

## Режими введення

- **Числа** — числа через пробіл, кому або крапку з комою
- **Довжина слів** — текст, рахуються довжини окремих слів
- **Довжина речень** — текст, рахуються довжини речень
- **Пошук символів** — підрахунок входжень заданих підстрядків у тексті (без емпіричної функції розподілу, моди, дисперсії)

## Стек
