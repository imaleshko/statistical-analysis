# Статистичний аналіз даних

Вебзастосунок для побудови варіаційного ряду, обчислення статистичних характеристик вибірки та кореляційного аналізу сусідніх літер у тексті.

## Можливості

Для режимів «Числа», «Довжина слів», «Довжина речень», «Пошук символів»:

- варіаційний ряд
- частоти `nᵢ`, відносні частоти `wᵢ`
- накопичувальні частоти `nᵢ нак`, `wᵢ нак`
- полігон частот

Додатково для режимів «Числа», «Довжина слів», «Довжина речень»:

- емпірична функція розподілу `F*(x)` — формула та графік
- мода, медіана, середнє
- дисперсія, середнє квадратичне відхилення
- початковий момент заданого порядку

Для режиму «Кореляція»:

- матриця частот сусідніх пар літер
- частоти по `X` та `Y`
- середні, СКВ і коефіцієнт кореляції `r`

## Режими введення

- **Числа** — числа через пробіл, кому або крапку з комою
- **Довжина слів** — рахуються довжини окремих слів
- **Довжина речень** — рахуються довжини речень
- **Пошук символів** — підрахунок входжень заданих підстрядків
- **Кореляція** — кореляція між сусідніми літерами за їх позицією в українському алфавіті
